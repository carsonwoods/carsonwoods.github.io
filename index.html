DigitalID

Status: In Development

This is my attempt to create a digital identity validation system.
Creates a need based extensible authentication system for validating the creator of a post.
Similar to a GPG key, but with a more layman usable implementation.
This project is still being actively developed and is in the early planning stage.
Code will become available at the link below when the initial alpha version is launched.
Finished project will be a part of the Atreides Corp.

HPC Research at Sandia National Labs

Status: Completed

Completed as a part of an internship at Sandia National Labs.
Contributed to the Spack Package Manager by adding multi-user support.
The feature exists as an open pull request on the main Spack repository.
In addition to the feature, a paper was written on the merits of the feature.
These merits were validated on Astra, an ARM based HPC platform at Sandia National Labs.
The paper was accepted in the HPCSYSPROS Workshop at Supercomputing '19.
